# Kane Skills

Kane 创建的技能库。

本仓库是一个公开技能库，包含独立封装的技能包，供其他用户安装、检视、复制与适配。

## 技能列表

| 技能                          | 使用场景                         | 输出                                 |
| --------------------------- | ---------------------------- | ---------------------------------- |
| `prompt-optimizer`          | 将模糊需求转化为可直接复制使用的提示词。         | 一个 `.md` 或 `.xml` 提示词文件。           |
| `request-refactor-plan`     | 将重构想法转化为小步提交的实施计划。           | 重构 RFC / GitHub issue 正文。          |
| `article-analyzer`          | 分析文章、论文、报告或长篇论述。             | 一组 Markdown 分析文件及 `99-summary.md`。 |
| `blog-checker`              | 审阅中文技术博客文章。                  | 结构化诊断审阅。                           |
| `skill-evaluator`           | 为智能体技能构建可复用的评估方案。            | 评估计划、评分模板与报告结构。                    |
| `skill-optimizer`           | 审计并改进智能体技能。                  | 审计报告、评估计划、改进提案与验收标准。               |
| `subtext-article`           | 将字幕、自动语音识别输出或转录文本转换为忠实的中文文章。 | 包含规范化转录、草稿、自检与终稿的完整文件夹。            |
| `merge-drafts`              | 将多份草稿合并为一篇润色后的文章。            | 最终合并文章及合并报告。                       |
| `concept-fable`             | 通过三段式中文寓言解释高阶概念。             | 故事、揭示、理论映射、边界与阅读方向。                |
| `douban-dice-review`        | 用六枚理论骰子撰写简洁的豆瓣式影评。           | 骰子总结与六句影评。                         |
| `session-achieve`           | 复盘多轮对话并提取可复用的提示词。            | 会话复盘与提示词经验。                        |
| `it-system-skill-distiller` | 将 IT 业务系统提炼为智能体可读的能力包。       | 经过校验的 `distilled/` 包结构。            |

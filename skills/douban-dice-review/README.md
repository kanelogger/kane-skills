# douban-dice-review

Use six theory dice to generate a short Chinese Douban-style film review.

## Use When

- The user asks for a 豆瓣影评, 骰子影评, 六个骰子写影评, or a theory-keyword film review.
- The review should be compact, stylized, and shaped by random theory keywords.
- Movie facts can be lightly checked when needed, but the core output is the six-sentence review.

## Output

- One dice roll summary generated by `scripts/roll_dice.js`.
- A six-sentence Chinese review, usually without numbering, following the dice order.

## Public Notes

The dice script is deterministic enough to audit the selected keywords while preserving randomness for each run.

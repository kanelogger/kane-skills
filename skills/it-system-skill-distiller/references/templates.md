# 蒸馏文档模板

这些模板是起点，不是完成品。复制后必须用需求、Schema、源码、API 文档和实跑验证中的事实替换占位符。

## README.md

```markdown
# <系统名> · 蒸馏技能包

本目录把 <系统名> 蒸馏为一套 AI 可理解、可检索、可调用的业务能力说明。AI 只通过 API 操作业务，不直接访问数据库。

## 阅读顺序 / 加载策略

常驻加载：
1. `00-overview.md`
2. `01-glossary.md`
3. `domain/_index.md`
4. `api/_index.md`

按需检索：
- `domain/*.md`
- `api/<domain>/*.md`
- `reference/schema/*.md`
- `reference/enums.md`
- `reference/error-codes.md`
- `reference/model-code-conflicts.md`
```
